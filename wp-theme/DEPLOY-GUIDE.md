# mixhost WordPress デプロイ手順

## 1. mixhostにWordPressをインストール

1. mixhostのコントロールパネル（cPanel）にログイン
2. 「WordPress」または「簡単インストール」をクリック
3. 以下を設定：
   - **インストール先URL**: `mgoro.net`（サブディレクトリなし）
   - **サイト名**: `Mゴロー E34 535i`
   - **管理者ユーザー名**: 任意（admin以外を推奨）
   - **管理者パスワード**: 強いパスワードを設定
   - **メールアドレス**: 自分のメールアドレス
4. 「インストール」をクリック

## 2. テーマをアップロード

### 方法A: ZIPでアップロード（簡単）

1. `mgoro-e34` フォルダをZIPに圧縮
   ```
   ターミナルで実行:
   cd /Users/pismo/Desktop/mgoro/HTML/wp-theme
   zip -r mgoro-e34.zip mgoro-e34/
   ```
2. WordPress管理画面 → 外観 → テーマ → 「新しいテーマを追加」
3. 「テーマのアップロード」→ ZIPファイルを選択 → 「今すぐインストール」
4. 「有効化」をクリック

### 方法B: FTPでアップロード

1. FTPクライアント（FileZillaなど）でmixhostに接続
2. `/public_html/wp-content/themes/` に `mgoro-e34` フォルダをアップロード
3. WordPress管理画面 → 外観 → テーマ → 「Mgoro E34 535i」を有効化

## 3. 初期設定

### パーマリンク設定
1. 管理画面 → 設定 → パーマリンク
2. 「投稿名」を選択
3. 「変更を保存」をクリック（2回押すと確実）

### トップページ設定
1. 管理画面 → 固定ページ → 「新規固定ページを追加」
2. タイトルを「ホーム」にして公開
3. 管理画面 → 設定 → 表示設定
4. 「固定ページ」を選択 → ホームページ: 「ホーム」
5. 保存

## 4. 既存記事の移行

### メンテナンス記事の追加方法

1. 管理画面 → メンテナンス → 「新規追加」
2. タイトルを入力（例: 「エンジンオイル交換」）
3. 本文に既存HTMLの内容をコピー＆ペースト
   - 画像は「メディアを追加」ボタンからアップロード
   - またはドラッグ＆ドロップで直接挿入
4. 右パネルで「メンテナンスカテゴリ」を選択（エンジン、足回り等）
5. 「公開」ボタンをクリック

### オフラインMTG記事の追加方法

1. 管理画面 → オフラインMTG → 「新規追加」
2. タイトルを入力（例: 「2015年11月 箱根ミーティング」）
3. 本文に内容を入力
4. 右パネルで「開催年」を選択
5. 「公開」をクリック

## 5. 日常の更新方法（初心者向け）

### 新しい記事を追加する
1. `https://mgoro.net/wp-admin/` にログイン
2. 左メニューから追加したい種類を選択：
   - メンテナンス記事 → 「メンテナンス」
   - オフラインMTG → 「オフラインMTG」
   - その他 → 「コンテンツ」
3. 「新規追加」をクリック
4. タイトルと本文を入力
5. 画像はエディタにドラッグ＆ドロップするだけ
6. カテゴリを選択して「公開」

### 記事を編集する
1. 管理画面で該当記事を開く
2. 内容を修正
3. 「更新」ボタンをクリック

- **SSL**: mixhostの無料SSL（Let's Encrypt）を有効にしてください
